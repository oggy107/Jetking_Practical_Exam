Q -> How to add and switch main network to Goreli test network in
Metamask?

1. click on metamask extension. This will open metamask window.
2. click on profile button > settings > advanced. This will open advanced settings pannel.
3. toggle on "show test networks" button and make sure it is enabled.
4. now you will be able to see test networks in your networks pannel.
5. click on "Goerli test network" to switch to goerli test network.
6. now you have successfully switched to goerli test network.
